# Privacy Policy / プライバシーポリシー

**Last Updated / 最終更新日: January 4, 2026**

## English

### Overview

"YouTube Live PIP with Chat" is a browser extension that enables Picture-in-Picture mode for YouTube Live videos with an accompanying chat window. This extension is designed with user privacy as a top priority.

### Data Collection

**This extension does NOT collect, store, or transmit any user data.**

Specifically, we do not collect:
- Personal information
- Browsing history
- Video watching history
- Chat messages or content
- Any other user data

### How the Extension Works

This extension operates entirely locally within your browser:
- It only activates on YouTube.com pages
- It uses the browser's native Picture-in-Picture API
- It opens YouTube's official chat popout window
- No data is sent to external servers

### Permissions Used

| Permission | Purpose |
|------------|---------|
| `activeTab` | To access the current YouTube tab and enable PIP functionality |
| `scripting` | To inject the PIP button into YouTube's video player |
| `host_permissions` (youtube.com) | To operate on YouTube pages |

### Third-Party Services

This extension does not use any third-party analytics, tracking, or advertising services.

### Changes to This Policy

If we make changes to this privacy policy, we will update the "Last Updated" date above.

### Contact

If you have any questions about this privacy policy, please open an issue on our GitHub repository.

## 日本語

### 概要

「YouTube Live PIP with Chat」は、YouTube Liveの動画をチャットウィンドウと共にピクチャーインピクチャーモードで表示するブラウザ拡張機能です。この拡張機能はユーザーのプライバシーを最優先に設計されています。

### データ収集について

**この拡張機能は、ユーザーデータを一切収集、保存、送信しません。**

具体的に、以下のデータを収集しません：
- 個人情報
- 閲覧履歴
- 動画視聴履歴
- チャットメッセージやコンテンツ
- その他のユーザーデータ

### 拡張機能の動作

この拡張機能は完全にブラウザ内でローカルに動作します：
- YouTube.comのページでのみ動作します
- ブラウザのネイティブPicture-in-Picture APIを使用します
- YouTubeの公式チャットポップアウトを開きます
- 外部サーバーへのデータ送信は行いません

### 使用する権限

| 権限 | 目的 |
|------|------|
| `activeTab` | 現在のYouTubeタブにアクセスしPIP機能を有効にするため |
| `scripting` | YouTubeの動画プレイヤーにPIPボタンを追加するため |
| `host_permissions` (youtube.com) | YouTubeページで動作するため |

### サードパーティサービス

この拡張機能は、サードパーティの分析、トラッキング、広告サービスを一切使用しません。

このプライバシーポリシーに変更を加える場合は、上記の「最終更新日」を更新します。
